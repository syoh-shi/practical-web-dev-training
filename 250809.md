# HTML Basics

## Heading elements

### Syntax

```html
<h1></h1>
```

* `<h1>` : opening tag
* `</h1>` : closing tag

**Element example:**

```html
<h1>Hello</h1>
```

### Available levels

* `<h1>` \~ `<h6>` exist; anything above level 6 does not exist.

### Rules

* Only one `<h1>` tag.
* Don’t skip levels (e.g., don’t put `<h3>` tags immediately after `<h1>` tags).
